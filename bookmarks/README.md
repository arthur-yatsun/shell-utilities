# shell_bookmarks

A minimalist utility for managing shell bookmarks. 

## Installation
1. Run ./install.sh script

## How it works
| Command         | Description     |
|-----------------|-----------------|
| `sb -c NAME`    | Create a bookmark called NAME for the current dir            |
| `sb -r`         | Remove bookmark for the current dir        |
| `sb -l`         | List all bookmarks        |
